# 📥 Instalação do OpenPLC no Windows

Este tutorial descreve o passo a passo para instalar o **SCADA-LTS** no Windows.

---

##  1 - Pré-requisitos

- Sistema operacional: Windows 10 ou 11
- Necessário estar utilizando o usuário Administrador

---

## 2 - Instalação do OpenPLC

1 - [Baixe o instalador do openPLC aqui](https://autonomylogic.com/download)

2- Clique em download como mostra a imagem a baixo
